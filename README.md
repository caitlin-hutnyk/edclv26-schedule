# EDC Las Vegas '26 — Set Times

Personal schedule app for EDC Las Vegas 2026 (May 15–17, Las Vegas Motor Speedway).

Built with React + TypeScript + Vite, deployed via GitHub Pages.

## Dev

```sh
npm install
npm run dev          # http://localhost:5173/edclv26-schedule/
npm run dev -- --host  # also exposes on local network for phone preview
```

## Deploy

```sh
npm run deploy       # builds and pushes to gh-pages branch
```

Site lives at https://caitlin-hutnyk.github.io/edclv26-schedule/
